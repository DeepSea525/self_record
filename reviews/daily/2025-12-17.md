### 问题2：三个飞轮都没转起来

#### 🔵 飞轮1：功能迭代飞轮

```
用户调研 → 挖掘功能 → 功能上线 → 功能复盘 → 挖掘新功能
```

**卡点**：受限于研发资源

#### 🔵 飞轮2：参考图生产飞轮

```
自动化获取素材 → 生产参考图 → 上线 → 观察反馈 → 动态调整
```

**卡点**：完整流程没有跑通

#### 🔵 飞轮3：图片效果飞轮

```
从参考图知道用户喜欢什么 
+ 从付费数据知道用户为什么买单 
→ 分析总结 → 优化效果
```

**卡点**：中间的分析总结很难做，找不到规律

### 员工思维 vs 老板思维

| 维度 | 员工思维 | 老板思维 |
|------|----------|----------|
| 核心问题 | 怎么快点完成这个任务？ | 这件事的价值是什么？ |
| 关注点 | 满足老板期待 | 用户付费、链路设计 |
| 视角 | 任务导向 | 价值导向 |

**例子**：

- 老板会想：商品详情图做成什么样是有价值的？用户付费的链路是什么？
- 大头兵会想：这个任务派给我了，怎么快点实现？

**问自己**：日常工作中，怎么帮助自己从员工思维转化到老板思维？

> 低头做事容易，抬头看路难。复盘的价值就是强迫自己抬头。
