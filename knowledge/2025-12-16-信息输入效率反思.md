# 信息输入效率：审视与反思

> 记录时间: 2025-12-16 22:35
> 分类: 方法论/认知
> 标签: #信息管理 #效率 #复盘方法

## 核心观察

每天/每周接收的信息，有多少是真的和**我要做的事**相关？

> 很多时候仔细复盘，发现很多东西其实跟你真正要做的事没多大关系。

## 可执行的方法

### 信息归因分析

每次记录信息时，问自己：

- 这条信息服务于哪个**项目**？
- 还是纯粹的**兴趣驱动**？
- 还是**无目的的浏览**？

### 周度信息审计

在周复盘中加入：

| 信息来源 | 数量 | 与项目相关 | 有效率 |
|----------|------|-----------|--------|
| 播客/文章 | | | |
| 社交媒体 | | | |
| 工作交流 | | | |

## 延伸思考

- 是否需要主动**减少无目的浏览**？
- 如何让信息输入**项目驱动化**？
