# 我用AI给自己搭了一套「个人成长操作系统」

今天做了一件有意思的事——用AI帮我搭建了一套个人记录和复盘系统。

## 起因

最近一直在想一个问题：

> AI Agent那么火，ReAct模型（推理→行动→反馈→迭代）被用在各种AI产品里。
>
> 那**人的成长，是不是也可以套用这个模型？**

想了想，答案是肯定的：

- **Observe**：接收信息输入
- **Act**：执行行动
- **Reflect**：复盘反思
- **Reason**：调整规划，迭代方法

## 于是

我让AI帮我搭了这样一套系统：

```
📁 inbox/         → 快速记录，AI自动分类
📁 areas/         → 长期关注的领域（工作/学习/生活/健康）
📁 projects/      → 有明确目标的项目
📁 reviews/       → 每日/每周/每月复盘
📁 knowledge/     → 方法论沉淀
```

还做了个简单的Web界面，可以可视化查看。

## 核心思路

记录分两种：

1. **项目驱动型**：为了某个目标而学/做的东西
2. **纯粹沉淀型**：信息积累，没有明确目的

每次记录时，问自己：这条信息是为了什么项目？还是单纯的兴趣浏览？

**一周后复盘**：有多少信息输入是真的和我的目标相关？

## 准备试用一周

接下来一周，我会：

- 每天记录信息输入
- 标注每条信息的关联度
- 周末深度复盘

如果有效，再来分享结果。
