# 项目：AI套图

> 创建日期: 2025-12-16
> 更新日期: 2026-05-11 (由 AI商品图 升级更名)

## 🎯 项目目标

### 愿景

通过AI技术提升成套图片的生成效率和质量，覆盖各类商业场景，实现从“单张商品图”到“行业套图”的全面演进。

### 项目模块

| 模块 | 描述 | 状态 |
|------|------|------|
| 套图制作 | 核心功能，支持龙虾对话式与经营包GUI生成套图 | 🔵 进行中 |
| 模板库建设 | 建立覆盖不同行业的模板库与底层知识库调用逻辑 | 🔵 进行中 |
| AB测试基建 | 建设图片AB能力，寻找能真正提升商业转化的套图 | ⬜ 规划中 |
| 行业拓展 | 将能力扩展到更多行业场景 | ⬜ 规划中 |
| 制图效果提升 | 优化生成质量和细节表现 | ⬜ 规划中 |

### 成功标准

- [ ] 模板库整体方案设计落地，跑通素材调用逻辑
- [ ] 龙虾与经营包的套图生图流程跑通并上线
- [ ] 建立起AB测试机制验证图片商业价值
