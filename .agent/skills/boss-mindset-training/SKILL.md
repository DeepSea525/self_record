# 老板思维训练

在每周复盘时，先进行这套"老板思维训练"，培养价值导向的思考习惯。

## 步骤

### 1️⃣ 假装你是业务负责人（5分钟）

问自己：

- 如果这是我的业务，我最关心什么指标？
- 当前业务最大的问题是什么？
- 本周我做的事情，对这个指标有帮助吗？

### 2️⃣ 回顾本周任务的"价值层"（5分钟）

列出本周的主要任务，对每个任务问：

| 任务 | 这件事的价值是什么？ | 如果不做会怎样？ |
|------|---------------------|------------------|
| | | |
| | | |

### 3️⃣ 记录"老板问过的问题"（3分钟）

本周老板在会议/沟通中问过哪些问题？

- 问题1：
- 问题2：

这些问题背后关心的是什么？

### 4️⃣ 下周计划的价值检验（2分钟）

看看下周计划做的事：

- 哪些是"任务导向"的？（因为老板说了要做）
- 哪些是"价值导向"的？（因为对业务有帮助）

能不能把更多时间花在"价值导向"的事上？

## 产出

完成后在周复盘中添加"老板思维训练"小节，记录：

1. 本周最关心的指标
2. 发现的价值盲区
3. 下周要刻意练习的点

## 周期

- **频率**：每周一次
- **时机**：周复盘时进行
- **时长**：约15分钟
