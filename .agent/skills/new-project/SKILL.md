# 创建新项目

当用户说"开始一个新项目"或"创建项目xxx"时，按以下步骤处理：

## 步骤

1. **获取项目信息**
   询问用户：
   - 项目名称
   - 项目目标（一句话描述）
   - 预计完成时间（可选）

2. **创建项目目录**
   在 `projects/` 下创建项目文件夹
   目录名格式：项目名称（使用中文或英文，保持简洁）

3. **初始化项目文件**
   使用模板创建：
   - `goal.md` - 基于 `templates/project-goal.md`
   - `log.md` - 基于 `templates/project-log.md`
   - `review.md` - 空文件，项目结束时填写

4. **替换模板变量**
   - `{{PROJECT_NAME}}` → 项目名称
   - `{{DATE}}` → 当前日期

5. **确认创建**
   告知用户项目已创建，并展示项目结构

## 项目状态管理

每个项目可以有以下状态：

- 🔵 进行中
- ⏸️ 暂停
- ✅ 已完成
- ❌ 已放弃

状态记录在 `goal.md` 的头部
