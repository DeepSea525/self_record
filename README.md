# 🧠 个人记录与复盘系统

> 基于 ReAct 模型的个人成长系统
>
> **Observe → Act → Reflect → Reason → 迭代**

## 📂 目录结构

```
自我记录/
├── 📁 inbox/              # 快速记录收件箱（AI自动分类）
├── 📁 areas/              # 责任领域（持续关注的方向）
│   ├── 工作/
│   ├── 学习/
│   ├── 生活/
│   └── 健康/
├── 📁 projects/           # 项目（有明确目标和终点）
├── 📁 reviews/            # 复盘存档
│   ├── daily/             # 每日复盘
│   ├── weekly/            # 每周复盘
│   └── monthly/           # 每月复盘
├── 📁 knowledge/          # 知识沉淀/方法论
└── 📁 templates/          # 模板库
```

## 🚀 快速开始

### 记录内容

直接告诉我你想记录什么，例如：

- "今天上午开了个会，讨论了xxx..."
- "学到了一个新概念..."
- "突然想到一个点子..."

我会帮你自动分类并归档。

### 创建复盘

告诉我"要复盘了"，我会帮你创建对应的复盘文档。

### 新建项目

告诉我"开始一个新项目"，我会帮你初始化项目结构。

## 🔄 ReAct 循环

### 输入阶段 (Observe)

```
                    📥 输入/记录
                         │
            ┌────────────┴────────────┐
            ▼                         ▼
   🎯 项目驱动型                📚 纯粹沉淀型
   (为了某个目标)              (信息积累)
            │                         │
            ▼                         ▼
   projects/xxx/             areas/ 或 knowledge/
```

### 执行与复盘阶段 (Act → Reflect → Reason)

```
   🎯 项目执行                    📚 沉淀积累
        │                              │
        ▼                              │
   记录到 log.md                       │
        │                              │
        ▼                              │
   执行后复盘                    ──────┘
        │                     （可能转化为项目输入）
        ▼
   调整规划迭代
```

## 📝 命名规范

| 类型 | 格式 | 示例 |
|------|------|------|
| 日常记录 | `YYYY-MM-DD-描述.md` | `2024-12-16-项目思考.md` |
| 每日复盘 | `YYYY-MM-DD.md` | `2024-12-16.md` |
| 每周复盘 | `YYYY-W周.md` | `2024-51周.md` |
| 每月复盘 | `YYYY-MM.md` | `2024-12.md` |
| 项目文件夹 | 项目名称 | `内容创作计划/` |

## 🏷️ 标签体系（建议）

- `#工作` `#学习` `#生活` `#健康` - 领域标签
- `#想法` `#复盘` `#计划` `#记录` - 类型标签
- `#重要` `#待办` `#已完成` - 状态标签
